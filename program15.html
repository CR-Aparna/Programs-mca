Predefined String and
math functions in Javascript

String Functions

  • toUpperCase(): Converts a string to uppercase.
    • Example: 'hello'.toUpperCase() → 'HELLO'
  • charAt(): Returns the character at a specified index.
    • Example: 'hello'.charAt(1) → 'e'
  • includes(): Checks if a substring exists within the string.
    • Example: 'hello world'.includes('world') → true

Math Functions

  • Math.round(): Rounds a number to the nearest integer.
    • Example: Math.round(4.7) → 5
  • Math.sqrt(): Returns the square root of a number.
    • Example: Math.sqrt(16) → 4
  • Math.max(): Returns the largest of zero or more numbers.
    • Example: Math.max(10, 20, 30) → 30
